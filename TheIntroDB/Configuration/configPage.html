TheIntroDB

Segment data (intro, recap, credits, preview) is fetched from TheIntroDB API using each item's TMDB or IMDB ID. Segments are not loaded when you press play; run Dashboard - Scheduled Tasks - TheIntroDB Media Segment Scan to populate them. Items need TMDB/IMDB IDs (e.g. from metadata) for the plugin to find segments.

Optional Bearer token; when set, your pending and accepted submissions are weighted higher in results.

When enabled, items that already have segments are skipped to avoid refetching.
To ensure fair use, TheIntroDB API has rate limits. If you have a large library, scanning may be temporarily paused to respect these limits. The process will automatically resume after a cool-down period. Please be patient while your library is scanned.
