TheIntroDB

Segment data (intro, recap, credits, preview) is fetched from TheIntroDB API using each item's TMDB or IMDB ID. Segments are not loaded when you press play; run the TheIntroDB media segment scan scheduled task to populate them.

Optional Bearer token; when set, your pending and accepted submissions are weighted higher in results.
All media
Whitelisting a library scans all movies and episodes inside it. Selecting individual shows only scans those Emby series. Any media outside the selected libraries or shows is left unchanged.
